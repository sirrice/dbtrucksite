Add a new table

Add the data at a URL to the database. It can be the URL of a csv file, json file, or HTML file that contains a table.

Explain how to use this form

For example, let's say we found a table on a website that we'd like to analyze:

Copy the URL (http://www.health.ny.gov/statistics/chac/cancer/ca_ora.htm) into the input box below, then give the data a name (e.g., oral_cancer).

The table will begin importing and you should be able to see it in the "all data" tab!
