Left Section

This column lists all of the tables in the database.

The numbers next to a table (e.g., table_name (50/100)) specifies that the table contains 100 rows and we extracted location information from 50 of the rows.

You can display a table's data in the main section by clicking on the table name.

 

Main Section

This section will list the tables that have been selected. We display the first 10 rows in each table.

Below each table is a form that allows you to specify any location related columns in the table. This information will be used to automatically geocode the table.

In addition, you can specify if all of the data in the table belongs to one city/zipcode/state. This may happen if you downloaded e.g., New York health data. If this is the case, the system cannot propery geocode the table without this information (e.g., "Springfield" is found is multiple states in the US ).
